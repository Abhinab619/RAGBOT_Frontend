# BackendMini React Frontend

Simple React client that posts a prompt to the FastAPI backend and renders the returned answer.

## Setup

1. Copy `.env.example` to `.env` (optional).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```

Requests are sent to `https://nsbot.online/aman`.
